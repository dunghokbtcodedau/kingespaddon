# 👑 King ESP Addon

> A Meteor Client addon for Minecraft 1.21.11 featuring powerful ESP (Extra Sensory Perception) modules.

---

## 📋 Requirements

| Dependency | Version |
|---|---|
| Minecraft | `1.21.11` |
| Meteor Client | `1.21.11-SNAPSHOT` |
| Fabric Loader | `0.18.2` |
| Java | `21+` |

---

## 📥 Installation

1. Install [Fabric Loader](https://fabricmc.net/use/) for Minecraft `1.21.11`
2. Install [Meteor Client](https://meteorclient.com/) (snapshot build for `1.21.11`)
3. Download the latest `.jar` from the [Releases](https://github.com/dunghokbtcodedau/kingespaddon/releases) page
4. Place the `.jar` file into your `.minecraft/mods/` folder
5. Launch Minecraft and enjoy!

> Alternatively, download `addonking.zip` from this repository and extract manually.

---

## 🔨 Building from Source

```bash
# Clone the repository
git clone https://github.com/dunghokbtcodedau/kingespaddon.git
cd kingespaddon

# Build with Gradle
./gradlew build
```

The output `.jar` will be located at `build/libs/`.
